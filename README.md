Movie Booking

- Movie Booking là một ứng dụng đặt vé xem phim trực tuyến, hỗ trợ cả nền tảng web và mobile, giúp người dùng dễ dàng tìm kiếm phim, đặt vé và theo dõi lịch chiếu.

🚀 Tính năng chính

+ Tìm kiếm phim: Hỗ trợ tìm kiếm theo tên phim.

+ Xem chi tiết phim: Hiển thị trailer, thông tin diễn viên, thời lượng.

+ Chọn ghế và đặt vé: Hiển thị sơ đồ ghế ngồi, chọn ghế và thanh toán vé.

+ Lưu trữ dữ liệu: Sử dụng Redux để tối ưu hóa việc lưu trữ dữ liệu phim.
